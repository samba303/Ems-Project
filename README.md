# Employee Management System (Full Stack Project)

# Project Overview
This is a Full Stack Employee Management System built using **Spring Boot (Java)** for backend and **React JS** for frontend.  
It allows users to manage employee data with authentication and CRUD operations.



##  Tech Stack

### Backend:
- Java
- Spring Boot
- Spring Security 
- Hibernate 
- REST APIs
- MySQL

# Frontend:
- React JS
- Bootstrap
- HTML, CSS, JavaScript
- Axios

# Database:
- MySQL


#  Features

- Add new employees
- Update employee details
- Delete employees
- View all employees
- User Login / Authentication
- REST API integration
- Responsive UI
